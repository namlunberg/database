Практика по работе с БД в PHP

1.
Пусть в базе данных есть такая таблица workers:
id	name	age	salary
1	Коля	23	400
2	Вася	24	500
3	Петя	25	600

    Необходимов вывести на страницу данную таблицу в виде тэга table, названия столбцов должны быть шапкой таблицы


2.
    Модифицируем предыдущую задачу, добавив возможность удаления работников.
Для этого добавим в таблицу еще одну колонку, в которой для каждого работника будет размещаться ссылка на удаление работника


3.
    Давайте теперь сделаем добавление нового работника с помощью формы. Вынесем форму добавления в отдельный файл, при добавлении сотрудинка, после отправки формы и перезагрузки страницы в форме должна остаться информация которую мы заполняли.

4.
    Доработаем код так, чтобы инпуты формы выводились с помощью функции, которая входным параметром принимает значение атрибута name поля. Таким образом в теле формы будет три вызова данной функции и вручную прописан input type="submit". Функционал сделанный в предыдущую задачу доджен остаться.

5. 
    Сделайте колонку 'редактировать' для каждого работника. Там должна быть ссылка, по переходу на которую появится страница с формой редактирования работника.

6.
    Добавить постраничную навигацию. Кол-во записей на странице можно взять, например, 2.

    Создайте переменные $a=10, $b=2 и $c=5. Выведите на экран их сумму.
    
    Дана строка. Проверьте, что она начинается на 'http://' или на 'https://'. Если это так, выведите 'да', если не так - 'нет'.

    Дан массив с элементами 'html', 'css', 'php'. С помощью функции implode создайте строку из этих элементов, разделенных запятыми.

    С помощью preg_match узнайте является ли строка числом, длиной до 12 цифр.

    Дана строка со звездочками 'aaa * bbb ** eee *** kkk ****'. Замените на '!' только двойные звездочки, но не одиночные или тройные и более.
